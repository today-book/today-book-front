# 🎄 크리스마스 책 추천 서비스

당신의 기분에 맞는 완벽한 책을 추천해드리는 크리스마스 특별 서비스입니다!

## ✨ 주요 기능

### 📱 모바일 최적화
- 모바일 환경에 완벽하게 최적화된 UI/UX
- 반응형 디자인으로 모든 스마트폰에서 원활한 사용
- 터치 인터랙션 최적화

### 🎅 크리스마스 테마
- 눈 내리는 애니메이션 효과
- 빨강, 녹색, 금색의 크리스마스 컬러 팔레트
- 따뜻하고 즐거운 크리스마스 분위기

### 🔍 스마트 추천 시스템
- 검색창을 통한 자유로운 기분 입력
- 12가지 감정 키워드 제공 (행복한, 설레는, 우울한, 외로운, 화난, 평온한, 지친, 호기심, 로맨틱한, 모험적인, 따뜻한, 긴장된)
- 기분에 따른 맞춤형 책 추천

### 📚 추천 결과
- 책 이미지 (동적 생성)
- 상세한 책 정보 (제목, 저자, 출판사, 출간일)
- 추천 이유 설명
- 카카오 로그인을 통한 결과 저장 기능

## 🚀 사용 방법

1. **기분 입력하기**
   - 상단 검색창에 "지금 나의 기분"을 입력하거나
   - 하단의 키워드 버튼 중 하나를 선택하세요

2. **책 추천받기**
   - "🎁 책 추천받기" 버튼을 클릭하면
   - AI가 당신의 기분에 맞는 완벽한 책을 추천해드립니다

3. **결과 확인하기**
   - 추천된 책의 표지, 정보, 추천 이유를 확인하세요
   - 마음에 드는 책이 있다면 카카오 로그인으로 저장할 수 있습니다

4. **다시 추천받기**
   - "🎄 다시 추천받기" 버튼으로 메인 화면으로 돌아가
   - 다른 기분으로 또 다른 책을 추천받아보세요!

## 🎨 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - 고급 애니메이션 및 그라디언트

- **Canvas API** - 동적 책 표지 생성

## 🌟 주요 기능 상세

### 눈 내리는 효과
- 50개의 눈송이가 랜덤한 속도와 크기로 떨어집니다
- CSS 애니메이션으로 부드러운 효과 구현
- 성능 최적화로 모바일에서도 원활한 동작

### 키워드 선택
- 시각적 피드백 (활성화 시 금색으로 변경)
- 모바일 진동 피드백 지원
- Active 상태 시 그림자 효과

### 책 추천 알고리즘
- 12개의 감정별 맞춤 도서 데이터베이스
- 키워드 또는 자유 입력 텍스트 기반 매칭
- 한국 베스트셀러 및 명작 선정

### 동적 책 표지 생성
- Canvas API를 사용한 실시간 이미지 생성
- 그라디언트 배경 및 텍스트 렌더링
- 각 책마다 고유한 컬러 팔레트

### 카카오 로그인
- 카카오 공식 로고 및 브랜드 컬러 (#FEE500) 사용
- 메인 페이지와 결과 페이지 모두에서 접근 가능
- 향후 카카오 OAuth API 연동 준비됨

## 🎁 추천 도서 목록 (12권)

1. **행복한** - 작은 것들의 신 (아룬다티 로이)
2. **설레는** - 노르웨이의 숲 (무라카미 하루키)
3. **우울한** - 달러구트 꿈 백화점 (이미예)
4. **외로운** - 미드나잇 라이브러리 (매트 헤이그)
5. **화난** - 죄와 벌 (표도르 도스토옙스키)
6. **평온한** - 채식주의자 (한강)
7. **지친** - 코스모스 (칼 세이건)
8. **호기심** - 사피엔스 (유발 하라리)
9. **로맨틱한** - 오만과 편견 (제인 오스틴)
10. **모험적인** - 반지의 제왕 (J.R.R. 톨킨)
11. **따뜻한** - 빨강머리 앤 (루시 모드 몽고메리)
12. **긴장된** - 살인자의 기억법 (김영하)
